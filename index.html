How It Works

GitShop is a unique e-commerce solution that leverages GitHub for content management. Store owners can edit product information directly through the GitHub API by providing their access token.

For Customers

Browse and shop as you would on any other online store. Enjoy a seamless shopping experience with beautiful product displays and detailed information.

For Store Owners

Add ?token=YOUR_GITHUB_TOKEN to the URL to enter edit mode. Update product details, images, and descriptions directly through the interface.

Technical Details

When in edit mode, all changes are saved back to the GitHub repository via the API. This allows for version-controlled content management without a traditional backend.
